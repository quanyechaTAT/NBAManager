# 首页球队/球员导航跳转设计

**日期：** 2026-06-25
**目标：** 为首页的球队排名和球员得分榜添加点击跳转到详情页的功能

---

## 1. 功能描述

### 球队排名行
- 整行可点击，跳转到 `/teams/detail?name=<teamName>`
- 悬浮时显示向右箭头提示
- 悬浮效果：背景高亮 + 箭头出现

### 球员得分行
- 整行可点击，跳转到 `/players/detail?id=<playerId>`
- 悬浮时显示向右箭头提示
- 悬浮效果：背景高亮 + 箭头出现

### 详情页返回
- 球队详情页：添加"返回首页"按钮
- 球员详情页：添加"返回首页"按钮

---

## 2. 路由格式

| 页面 | 路由格式 |
|------|----------|
| 球队详情 | `/teams/detail?name=<teamName>` |
| 球员详情 | `/players/detail?id=<playerId>` |
| 返回首页 | `/dashboard` |

---

## 3. 数据字段

### 球队排名数据 (TeamRank)
- `teamName`: 球队名称（用于跳转）

### 球员得分数据
- 当前 `topScorers` 包含: `playerName`, `ppg`, `teamName`
- **需要确认是否包含 `id` 字段**
- 如果不包含 `id`，需要通过 `playerName` 查询或使用其他方式

---

## 4. 文件变更

| 文件 | 变更 |
|------|------|
| `DashboardView.vue` | 添加行点击事件、悬浮箭头样式 |
| `TeamDetailView.vue` | 添加返回按钮 |
| `PlayerDetailView.vue` | 添加返回按钮 |

---

## 5. 验收标准

1. 球队排名行可点击跳转到球队详情页
2. 球员得分行可点击跳转到球员详情页
3. 悬浮时显示向右箭头
4. 详情页有返回首页按钮
5. 从详情页返回后回到首页
